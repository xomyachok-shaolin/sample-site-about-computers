Задание и его выполнение. ЛР1.

Часть 1.

  1. Определить путевое имя рабочего каталога.
    pwd

    Как обозначается корневой каталог?
    /

    Какое путевое имя получили (относительное или абсолютное)?
    абсолютное

  2. Создать в начальном каталоге два подкаталога.
    mkdir cat1 | mkdir cat2

    Просмотреть содержимое рабочего каталога.
    ls

    Просмотреть содержимое родительского каталога, не переходя в него.
    ls ..

  3. Перейти в системный каталог.
    cd /tmp

    Просмотреть его содержимое.
    ls

    Просмотреть содержимое начального каталога.
    ls ..

    Вернуться в начальный каталог.
    cd ..

  4. Удалить созданные ранее подкаталоги.
    rm -i ~/cat1 ~/cat2 -r

  5. Получить информацию по командам ls и cd с помощью утилиты man.
    man ls cd

    Изучить структуру man-документа.
    man man

  6. Получить краткую информацию по командам ls и cd с помощью команды whatis и apropos.
    whatis ls cd; apropos ls cd

    В чем различие?
    Команда apropos выполняет ту же задачу, что и команда whatis но поиск выполняется по образцу, а не по целым словам.

Часть 2.

  1. Создайте в домашнем каталоге следующую структуру подкаталогов (существующие каталоги не удаляйте!)

    a. домашний каталог ----| cd /home
    1. |-ВашаФамилия-| mkdir asminaev
    a. |-1-| mkdir 1 4
    b. | |-2 mkdir 1/2 1/3
    c. | |-3
    d. |
    e. |-4

  2. Скопируйте файл /etc/group в каталог 1 используя абсолютные имена копируемого файла и каталога назначения.
    cp -i /etc/group /home/1

  3. Скопируйте файл /etc/group в каталог 2 используя абсолютное имя копируемого файла и относительное имя каталога назначения.
    cp -i /etc/group 1/2

  4. Скопируйте файл /etc/group в каталог 3 используя относительные имена копируемого файла и каталога назначения.
    cp -i ../etc/group 1/3

  5. Скопируйте файл /etc/group в каталог 4 используя абсолютные имена копируемого файла и относительное имя каталога назначения с использованием специального символа~.
    cp -i /etc/group ~/4

  6. При помощи одной команды зайдите в каталог 3.
    cd /home/1/3

  7. Удалите файл group из каталога 4 при помощи одной команды.
    ls ../../4/group

  8. Перейдите в свой домашний каталог. Удалите каталоги 1 и 4.
    cd /home; rm 1 4 -r

  9. Выведите первые и последние 13 строк файла /etc/group.
    cat /etc/group | head -n 15; cat /etc/group | tail -n 15
